13.2 移动端RT API

Vulkan Ray Query (推荐):

移动端主要使用Vulkan的Ray Query而非完整的Ray Tracing Pipeline。

与桌面端API的区别：

特性 桌面端 移动端
光线生成着色器 有 需手动实现
命中着色器 有 需手动实现
加速结构 完整支持 支持
灵活性 较低 高
性能开销 硬件优化 更依赖实现
